February 3, 2026
    Classic MONAN tutorial:
        1 - Entrar na jaci no seu diretorio de trabalho ( /p/projetos/monan_atm/madeleine.gacita)
        (aqui  você pode criar um diretório para realizar o teste)

        2 - Download dos scripts ( git clone https://github.com/monanadmin/scripts_CD-CT.git)

        3 - Entrar no diretório scripts_CD-CT e mudar para versão develop ( "cd scripts_CD-CT/"
        e depois "git checkout develop"). Após isso entrar no diretório "scripts"

        4 - Compilar o Monan  ( ./1.install_monan.bash
        https://github.com/monanadmin/MONAN-Model.git 1.4.3-rc 1.2.0 )
        --> instala o monan do github na versão 1.4.3 utilizando o convert_mpas 1.2.0

        Se tudo der certo, voce ja compilou o Monan na jaci. Pode verificar o executável no
        diretório execs (ls -l  ../execs).

        5 - Execução do Pre  ( ./2.pre_processing.bash ${EXP} ${RES} ${YYYYMMDDHHi} ${FCST} ).
        Exemplo resolução 10km e 24 horas forecast (  ./2.pre_processing.bash GFS  5898242
        2025102500  24 )

        5 - Execução do Model ( ./3.run_model.bash ${EXP} ${RES} ${YYYYMMDDHHi} ${FCST}).
        Exemplo resolução 10km e 24 horas forecast ( ./3.run_model.bash GFS  5898242
        2025102500  24 )

        6 - Execução do Post ( ./4.run_post.bash ${EXP} ${RES} ${YYYYMMDDHHi} ${FCST}).
        Exemplo resolução 10km e 24 horas forecast ( ./4.run_post.bash GFS  5898242  2025102500  24 )

        Voce pode verificar as saídas em dataout ( ls -l ../dataout)
