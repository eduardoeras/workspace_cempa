    GRADS Basics:
        grads -b                        batch mode
        grads -c "command"              run a command
        grads -b -c "script.gs"         run a script in bach mode

        quit                            exit grads
        reinit                          reset everithing (close files)
        reset                           reset settings (don't close files)
        clear                           clear plot canvas

        open file.clt                   opens a Control file 
        sdfopen file.nc                 opens a NetCDF file
        set dfile 1                     set default file as file 1
        close 1                         close file 1

        display var                     plot var to canvas
        gxprint file.png y1080 x1920    export png output

    Git commands (on synchronized directory):
        git status          Fundamental current status of git synchronization
        git pull            Download remote changes and merge

        git add <file>
        git commit -m <message>
        git push

        git restore --source=origin/develop /file_to_be_restored
        git checkout        change branch

        git remote -v       Show remote repositories
        git diff            List unsychronized changes
        git log             Show full commit history
        git log --oneline   Show short commit history
        git rm <file>       Remove tracked file
